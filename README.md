## Usage

run `npm i` to install dependencies / update packages.

then run `npm run *NAMEOFSERVER/CLIENT*` the server/client of your choice.

Servers/Clients:

- "login-server"
- "login-client"
- "gateway-server"
- "gateway-client"
- "zone-server"
- "zone-client"

You can modify the parameters of these servers in their respective Javascript files.
